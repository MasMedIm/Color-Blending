# To know !

This dependency was created just for fun and should NOT be in a production environment.

## Getting Started

```
npm i color-blending
```

### Importing


```
let blend = require('color-blending')
```

### How to use it ?

Pretty straightforward :

```
console.log(blend("#E6E6FA", "#FF69B4", "#B0C4DE"))
```
